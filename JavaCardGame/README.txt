Includes 3 test packs: cards1.txt, cards2.txt, cards3.txt.

Cards1.txt and cards3.txt contain 32 values each so test with 4 players.

Cards2.txt contains 40 values so test with 5 players.

To test, when prompted, enter the number of players e.g. if testing cards1.txt enter 4

Next you will be asked for the location of the pack, for this you can just enter the name of the pack you are trying to test, e.g. cards1.txt since the packs are within the same directory as the source files.
